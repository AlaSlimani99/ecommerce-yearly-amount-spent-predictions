# ecommerce-yearly-amount-spent-predictions

Project Description:

This project aims to predict the yearly amount spent by a user in an e-commerce business based on customer behavior and engagement data.
The goal is to build a baseline machine learning model that demonstrates the full ML workflow, from data preprocessing to model evaluation and business interpretation.


Workflow:

1-Data Loading & Cleaning

2-Exploratory Data Analysis (EDA)

3-Feature Selection & Preprocessing

4-Train / Test Split

5-Model Training (Linear Regression)

6-Model Evaluation

7-Business Interpretation of Results
